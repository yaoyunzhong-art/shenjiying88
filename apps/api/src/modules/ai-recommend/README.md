# AI Recommend 智能推荐

> AI 推荐引擎，提供热门推荐、个性化推荐与推荐策略管理

## 功能
- 热门推荐与个性化推荐
- 推荐策略配置
- 推荐结果生成

## API
- GET /ai-recommend/recommendations/popular — 热门推荐
- GET /ai-recommend/recommendations/personalized — 个性化推荐
- GET /ai-recommend/recommendations — 综合推荐
- POST /ai-recommend/generate — 手动生成推荐
- POST /ai-recommend/strategies — 策略管理
