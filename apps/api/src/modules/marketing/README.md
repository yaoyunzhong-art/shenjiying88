# Marketing 营销

> 综合营销管理服务，支持 RFM 分析、A/B 测试与营销活动

## 功能
- RFM 客户价值分析
- 客户分群
- A/B 测试
- 营销活动记录

## 依赖
- MarketingMetricsModule, AgentModule, TenantModule

## API
- POST /marketing/rfm/compute — 计算 RFM
- GET /marketing/rfm/stats — RFM 统计
- GET /marketing/rfm/segments — RFM 分群
- POST /marketing/ab/create — 创建 A/B 测试
- POST /marketing/ab/record — 记录 A/B 测试
