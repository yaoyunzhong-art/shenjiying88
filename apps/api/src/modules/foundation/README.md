# Foundation 基础服务

> 系统基础核心服务，提供引导、模块概览与告警目录

## 功能
- 系统引导数据
- 模块列表与概览
- 告警概览与目录

## 依赖
- TenantModule

## API
- GET /foundation/bootstrap — 引导数据
- GET /foundation/modules — 模块列表
- GET /foundation/overview — 系统概览
- GET /foundation/overview/alerts — 告警概览
- GET /foundation/overview/alerts/catalog — 告警目录
