## 知识总结

### 模式
1. **In-memory Store + Reset**: 测试使用 `resetFinance*TestState()` 函数清空内存数据
2. **ResponseInterceptor**: 统一将响应包装为 `{ success, data, message, timestamp }`
3. **报表生成**: 同步生成 + SSE 事件推送 (report.generating → report.completed / report.failed)
4. **对账报告导出**: `\ufeff` BOM + CSV 格式兼容 Excel
