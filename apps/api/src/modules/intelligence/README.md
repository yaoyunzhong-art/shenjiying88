# Intelligence 智能选址

> 智能选址与门店规划服务，支持可行性分析、财务概览与运营评估

## 功能
- 选址可行性分析
- 财务概览
- 门店选址评估
- 门店运营数据

## 依赖
- ScheduleModule, AgentModule

## API
- POST /intelligence/feasibility — 可行性分析
- POST /intelligence/finance-panorama — 财务概览
- GET /intelligence/siting-assessment — 选址评估
- POST /intelligence/store-planning — 门店规划
- GET /intelligence/operations/:storeId — 运营数据
