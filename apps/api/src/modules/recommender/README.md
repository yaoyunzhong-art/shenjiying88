# Recommender 推广大使

> 推广大使推荐系统，支持推荐、上下文查询、反馈与搜索

## 功能
- 推广大使智能推荐
- 上下文查询
- 反馈管理
- 大使搜索
- 统计数据

## 依赖
- KnowledgeModule, AgentModule

## API
- POST /recommender/recommend — 推荐
- GET /recommender/context/:championId — 上下文
- POST /recommender/feedback — 反馈
- POST /recommender/search — 搜索
- GET /recommender/stats — 统计
