# SaaS Advanced SaaS 高级功能

> SaaS 多租户高级功能管理，支持主账户、治理规则与批量管理

## 功能
- SaaS 高级管理
- 主账户管理
- 批量主账户设置
- 治理规则查询

## API
- POST /saas-advanced — 创建
- GET /saas-advanced — 列表
- GET /saas-advanced/primary/current — 当前主账户
- POST /saas-advanced/primary/batch/current — 批量主账户
- GET /saas-advanced/governance/active-without-primary — 无主账户治理
