# Device Usage Report 设备使用报告

> 设备使用统计报告服务

## 功能
- 设备使用报告管理
- 使用统计概览
- 报告 CRUD

## 依赖
- AgentModule, TenantModule

## API
- GET /device-usage-report — 报告列表
- GET /device-usage-report/summary — 统计概览
- GET /device-usage-report/:id — 报告详情
- POST /device-usage-report — 创建报告
- DELETE /device-usage-report/:id — 删除报告
