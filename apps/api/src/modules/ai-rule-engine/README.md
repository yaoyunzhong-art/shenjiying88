# AI Rule Engine AI 规则引擎

> 规则驱动的 AI 决策引擎，支持会员等级评估、设备异常检测与风险评分

## 功能
- 规则评估与推理
- 会员等级自动评估
- 设备异常检测
- 批量评估
- 综合风险评分

## 依赖
- AgentModule

## API
- POST /ai-rule-engine/evaluate — 规则评估
- POST /ai-rule-engine/evaluate/member-level — 会员等级评估
- POST /ai-rule-engine/evaluate/device-anomaly — 设备异常检测
- POST /ai-rule-engine/evaluate/batch — 批量评估
- POST /ai-rule-engine/evaluate/risk-score — 风险评分
