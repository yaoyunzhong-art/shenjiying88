# 🚶 巡检 / 遍历脚本 (Walks)

> M5 平台代码与功能完整性遍历脚本集，用于批量验证页面结构、接口覆盖和功能清单。

## 概述

本目录存放按模块/场景组织的 walkthrough 清单，以结构化表格逐项记录待验证的子页面、组件、测试覆盖与关键指标。每份 walkthrough 适用于人工或自动化巡检，可嵌入 CI 流水线作为质量门禁。

## 文件

| 文件 | 说明 |
|------|------|
| `stores-walkthrough.md` | 门店管理全流程遍历清单，覆盖 26 个子页面的 page.tsx、行数阈值、测试文件、'use client' 指令等 3 项验证标准 |

## 用法

walkthrough 文档可按如下流程使用：

1. **对照清单** — 逐项检查 page.tsx 是否存在、行数是否达标
2. **标记状态** — 在表格中标注 ✅/❌ 与实际缺陷
3. **驱动修复** — 将未通过的条目转为开发任务或圈梁测试用例

巡检结果可作为「圈梁测试」的前置输入，确保功能交付的完整性。
