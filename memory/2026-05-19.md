- **尝试 1**: 全局 PrismaService DI 重构 (30+文件) → agent 超时 (10min)
- **尝试 2**: 再次全局 DI → 158 回归失败 → git checkout 回退
- **尝试 3**: 第三次全局 DI → 超时
- **胜出方案**: **懒加载 (Lazy Init)**
  - 每个 service 加 private getter:
    ```
    private _prisma: any = null
    private get prisma() {
      if (!this._prisma) this._prisma = new PrismaClient()
      return this._prisma
    }
    ```
  - 只改 member.service.ts + booking.service.ts
  - 2行/文件，不改构造函数
  - 10min 超时但改完成，代码已 git push
