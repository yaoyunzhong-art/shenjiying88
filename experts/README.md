# 👥 专家模块 (Experts)

> M5 平台「40+4 专家团」角色定义库，覆盖 8 大领域的业务角色与 Champion 老板团队。

## 概述

本目录维护了 **44 个专家角色** 的详细定义文件（E1–E54），每个文件描述某一角色的职责范围、领域知识和与系统的交互模式。专家角色按领域分为 8 大板块+老板团队，支撑多租户、多市场场景下的业务决策与系统设计。

## 领域分组

| 领域 | 人数 | 范围 |
|------|------|------|
| 🏗️ 架构 | 3 | 系统架构、安全架构、硬件设计 |
| 🔒 安全 | 3 | 安全审计、运维、合规 |
| 📢 营销 | 10 | 营销策略、门店运营、法律合规等 |
| 📊 数据 | 2 | 数据分析、产品管理 |
| 💰 财务 | 3 | 财务、法务、投资 |
| 🎨 体验 | 3 | UX、客服、培训 |
| 🏢 租户 | 10 | 品牌/门店/会员/供应链角色 |
| 📦 后勤 | 7 | 物流、采购、SRE 等 |
| 👑 老板团队 | 5 | 董事长、总经理、区域等 Champion |

- `templates/` — 专家角色模板文件
