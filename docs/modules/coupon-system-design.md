# P-48 营销券系统详细设计文档

## 1. 系统概述

营销券系统是数字运动潮玩平台 V17 的核心营销子系统，支持多种优惠券类型、灵活的核销规则、跨门店使用场景，以及完整的生命周期管理。

### 1.1 核心能力

| 能力 | 描述 | 业务价值 |
|------|------|----------|
| **三级模型** | 券模板 → 券批次 → 券实例 | 灵活管理、精准发放 |
| **跨门店核销** | 多店通用、店间结算 | 提升用户体验 |
| **多级优惠** | 满减、折扣、免单组合 | 复杂营销场景支持 |
| **实时库存** | 券库存原子扣减 | 防止超发 |
| **核销风控** | 频率限制、黑名单 | 防止羊毛党 |
