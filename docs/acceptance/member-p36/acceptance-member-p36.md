## 1. 模块概述

会员中心是店A客户关系管理的核心子系统，覆盖会员注册、等级管理、积分累计/抵扣、余额充值/支付、消费流水查询等完整会员生命周期管理。本验收档覆盖 P-36 阶段全部 10 项功能需求的验收标准及测试场景。

**验收范围:**
- 会员注册与查询
- 会员等级展示与升级
- 积分累计与扣减
- 余额充值流程
- 余额支付与不足拦截
- 消费记录流水展示
- 会员权益展示
- 异常场景(重复注册/无效手机号)

## 2. 验收标准

### 2.1 会员注册与查询

#### AC-MEMBER-36-01: 新会员注册
- **Given** 手机号 `13800138001` 尚未注册
- **When** 收银员输入该手机号及姓名，点击"注册"
- **Then** 系统创建会员，等级=普通，积分=0，余额=0，向下发验证码

#### AC-MEMBER-36-02: 重复手机号注册拦截
- **Given** 手机号 `13800138000` 已存在
- **When** 使用该手机号再次注册
- **Then** 提示"该手机号已注册"，不创建新会员

#### AC-MEMBER-36-03: 手机号格式校验
- **Given** 注册页面打开
- **When** 输入非11位数字手机号
- **Then** 提示"请输入正确的11位手机号"，阻止提交

#### AC-MEMBER-36-04: 会员手机号查询
- **Given** 会员 `13800138000` 已注册(金卡/积分3280/余额¥450)
- **When** 收银员输入该手机号
- **Then** 展示: 姓名=张三、等级=金卡、积分=3,280、余额=¥450，响应<3s

#### AC-MEMBER-36-05: 未注册手机号查询
- **Given** 手机号 `13900000000` 未注册
- **When** 收银员输入该手机号
- **Then** 提示"该手机号未注册"，提供注册入口

### 2.2 等级与积分

#### AC-MEMBER-36-06: 等级展示与升级进度条
- **Given** 会员为银卡(已消费¥500)，下一级金卡需¥2000
- **When** 展示会员卡片
- **Then** 显示银卡标识 + 进度条"已消费¥500/¥2000 (25%)"

#### AC-MEMBER-36-07: 消费后积分自动累计(普通)
- **Given** 会员等级=普通(积分倍率1x)
- **When** 消费 ¥100，支付成功
- **Then** 积分 +100

#### AC-MEMBER-36-08: 消费后积分自动累计(金卡)
- **Given** 会员等级=金卡(积分倍率1.5x)
- **When** 消费 ¥100，支付成功
- **Then** 积分 +150

#### AC-MEMBER-36-09: 积分抵扣成功
- **Given** 会员积分≥¥500，当前应付¥50
- **When** 客户选择"用积分抵扣"
- **Then** 积分 -500 (按100积分=¥1扣减)，应付 -¥5

#### AC-MEMBER-36-10: 积分抵扣不足
- **Given** 会员积分=50 (<100)
- **When** 客户尝试用积分抵扣
- **Then** 提示"积分不足，无法抵扣"

#### AC-MEMBER-36-11: 自动等级升级
- **Given** 会员当前等级=普通，累计消费=¥480
- **When** 再消费¥20，累计消费达¥500
- **Then** 等级自动升级为银卡，弹窗提示客户升级

### 2.3 余额管理

#### AC-MEMBER-36-12: 余额充值
- **Given** 会员当前余额=¥0
- **When** 客户点击充值¥100，通过微信支付成功
- **Then** 余额变为¥100，生成充值流水记录

#### AC-MEMBER-36-13: 余额支付成功
- **Given** 会员余额=¥100，应付=¥50
- **When** 选择余额支付
- **Then** 余额变为¥50，订单完成，生成支付流水记录

#### AC-MEMBER-36-14: 余额不足拦截
- **Given** 会员余额=¥20，应付=¥50
- **When** 选择余额支付
- **Then** 提示"余额不足，请选择其他支付方式"，订单不生成

### 2.4 流水与权益

#### AC-MEMBER-36-15: 消费记录列表
- **Given** 会员近30天有5条充值/消费记录
- **When** 会员在小程序端点击"消费记录"
- **Then** 列表按时间倒序展示，每条显示: 类型/金额/时间/状态

#### AC-MEMBER-36-16: 权益展示
- **Given** 会员等级=金卡
- **When** 点击"查看权益"
- **Then** 展示金卡权益列表: 积分1.5x, 9折, 生日礼

#### AC-MEMBER-36-17: 会员到期提醒
- **Given** 会员有效期为2026-08-01，当前为2026-07-20
- **When** 会员登录小程序
- **Then** 展示"会员将于12天后到期"提示条
