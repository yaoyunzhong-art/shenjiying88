## 什么是ADR

对于跨模块架构变更(如P-31多租户RLS)，在写代码前先记录决策：
- 为什么选A方案不选B
- 影响哪些现有模块
- 接口合同锁定

## V17 ADR流程

### 触发条件 (任一满足)

```
1. 跨模块新增/修改接口
2. 数据访问层变更 (如RLS/TenantContext)
3. 安全架构变更
4. 第三方集成方案选型
5. 部署架构变更
```

### ADR步骤

```
G0 PRD定义 → 识别架构变更 → 写ADR → G1方案设计 → G3代码实现
              ↑                       ↓
              圈梁验证器检查ADR是否存在
```
