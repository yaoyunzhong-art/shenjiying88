# 门店每日打烊SOP

> 分类: 门店管理 | 标签: 打烊流程,每日SOP,安全巡检 | 适用: 门店运营,值班经理,服务团队

## 概述

每日打烊是电玩城运营中最容易被忽视但至关重要的一环。标准化的打烊SOP不仅关系到设备安全和财务安全，更直接影响第二天的营业效率和顾客体验。行业内因打烊不规范导致的设备损坏、现金丢失、安全隐患等事件占比约占总运营事故的28%。尤其是在深夜运营的电玩城（营业至23:00后），员工疲劳状态下更容易出现操作疏漏。

一套完善的门店打烊SOP应覆盖设备关机、现金清点、卫生清洁、安全检查、系统日结五个核心模块。按照shenjiying88的标准，每家门店的夜间打烊流程应控制在45分钟内完成，由值班经理全程监督并拍照留痕，确保每项操作可追溯。

## 核心内容

- **设备关闭与统计（闭店前30分钟）**: 逐台关闭游戏设备电源（非UPS直连设备需注意关机顺序避免跳闸）、记录各设备当日投币数/营收额/故障情况、清点彩票存量和奖品库存、关闭空调/排风/音响等辅助设备。彩票机需单独记录剩余票纸量。

- **现金与营收清点（闭店后15分钟内）**: 双人协作清点各收银位现金（纸币、硬币分类）、核对系统营收数据（偏差超过±2%需查明原因）、填写《每日营收交接表》、现金存入保险柜（双人密码+钥匙）、微信/支付宝收款需打印当日汇总单。

- **卫生清洁（闭店后15-30分钟）**: 地面深度清洁（游戏区/休息区/卫生间/走廊）、设备表面擦拭（触摸屏用专用清洁剂）、玻璃/镜面/奖品格清洁、垃圾清运。每周一次深度清洁（地毯洗护/空调滤网清洗/设备内部除尘）。

- **安全检查（闭店后30-45分钟）**: 确认所有电源插座已拔除或关闭（非24小时设备）、关闭水阀（卫生间/饮水区）、检查消防通道是否畅通、确认监控系统正常运行、启动夜间安防模式（门磁/红外/报警联动）。值班经理填写《夜间安全巡检表》。

- **系统日结与数据上报**: 运行POS系统日结操作、生成当日《门店运营日报》（含营收/客流/币耗/奖品兑换/设备故障等数据）、上传至总部管理系统、同步次日排班信息。如遇系统异常需手动备份数据并上报IT支持。

## 实际案例（基于电玩城场景）

广州某电玩城曾因值班员工在打烊时忘记关闭彩票机电源，导致彩票机夜间程序异常卡纸，次日早上发现时已造成约2,000元的彩票浪费。同时因未按规定双人清点现金，发现短款500元后无法追责。事后门店更新了打烊SOP，增加"逐项签收+拍照留痕"机制，并在每台设备旁张贴"关机确认贴纸"，此类问题再未发生。

## 可落地清单

- [ ] 制作门店《打烊SOP逐项确认卡》，每项操作完成后由操作人和监督人双签
- [ ] 安装夜班监控补光灯，确保闭店后30分钟内的操作在监控下进行
- [ ] 每月进行一次打烊流程演练（含突发事件模拟），演练记录纳入门店考核
