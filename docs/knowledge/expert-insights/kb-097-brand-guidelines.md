# 品牌使用规范

> 分类: 业务运营 | 标签: 品牌管理,品牌规范,VI体系 | 适用: 品牌部,市场部,门店管理

## 概述

品牌使用规范（Brand Guidelines）是连锁电玩城品牌一致性的"宪法"。当门店从1家扩展到5家、10家甚至更多时，如果没有统一的使用规范，"品牌"将因不同门店的随意使用而变得支离破碎——顾客在A店看到的Logo颜色和在B店不同、会员卡设计风格不统一、门店装修氛围各异。品牌一致性是建立品牌信任的基础，据调查，保持品牌一致性的企业营收平均比不一致企业高出23%。

电玩城品牌的特殊性在于，它不仅是"视觉识别系统"，更是一套"体验识别系统"——包括视觉（Logo/色彩/字体/空间设计）、听觉（背景音乐/喇叭提示音的调性）、行为（员工的服务用语、语调、仪态）、情绪（顾客在店内的整体感受）四个层面。品牌使用规范应覆盖所有与顾客接触的触点，从线上（公众号/小程序/美团/抖音店铺）到线下（门店装修/员工制服/物料设计/设备涂装）。

## 核心内容

- **品牌核心要素（Logo/颜色/字体/标识）**: Logo规范：明确最小使用尺寸（线上≥15mm，线下≥30mm）、安全区域（Logo四周保留Logo高度1/2的空白区域）、禁止行为（不可变形/拉伸/旋转/加阴影/更换颜色/放在复杂背景上）；品牌色系定义：主色（用于Logo/核心视觉，如活力橙/科技蓝）、辅助色（用于装饰/点缀/促销）、中性色（用于文字/背景）。需要给出CMYK（印刷用）、RGB（屏幕用）、HEX（网页用）三种格式的颜色值。字体规范：中文标题字/正文字/英文字体分别指定；禁用字体列表。

- **门店空间设计规范**: 这是电玩城品牌规范独特且重要的部分。核心内容：①外立面——门头招牌的材质/尺寸/灯光色温选择、品牌Logo的安装位置和高度；②收银区——收银台设计风格/高度/宽度、背后品牌墙的设计模板；③游戏区——各类型设备的分区规划（彩票区/街机区/VR区/娃娃机区）的标识系统、地面/墙面/天花板的色块分配；④兑奖区——彩票兑换柜台的设计规范、奖品展示的排列方式和灯光要求；⑤洗手间——卫生标准、装饰风格（品牌信息的融入方式）。每一个设计元素都应有具体的"可接受"和"不可接受"的示例对比。

- **线上渠道品牌规范**: 美团/大众点评店铺装修规范：Logo尺寸和位置、店铺头图和背景图的风格要求、菜品/活动照片的拍摄指南（构图/光线/色调）；公众号/小程序：头像/关注后回复/底部菜单/文章模板的品牌化一致性；抖音：头像/背景/视频封面的品牌元素使用、视频色调的后期标准；小红书：内容风格/排版调性/话题标签的规范建议。每一个线上触点都需要"标准化模板"（使用设计工具如Canva/Figma可快速生成）。

- **员工形象与行为规范**: 视觉层面——制服设计（款式/颜色/材质/配件/佩戴方式、不同岗位（导玩员/收银/技术/管理人员）的区分设计、工牌样式和佩戴位置；行为层面——接待标准用语（首句问候语/结束语/特殊场景用语的规范化）、微笑服务标准（与顾客眼神接触时保持微笑、语调友好而自信）、服务动作标准（指引手势/递送物品的方式/与顾客交谈的距离和站位）。员工是品牌最重要的"代言人"，员工形象/行为的一致性直接决定了顾客对品牌的感受。

- **品牌应用场景检查清单**: 每个需要品牌触达的触点都应列入检查清单，确保在执行中不漏掉任何一个。门店触点：进门招牌/收银台背景/设备顶贴纸/奖品展示架/会员卡/代金券/彩票/积分卡/员工工牌/购物袋/停车指示牌=共12个触点。线上触点：公众号横幅/小程序首页/美团店铺页面/抖音主页面/小红书账号首页/微信头像/App启动页/广告素材。每个触点指定"设计模板编号"和"制作要求"以便快速落地执行。

## 实际案例（基于电玩城场景）

某连锁电玩城品牌在扩张至第3家门店时发现：①A店的招牌使用了品牌主色，但B店因为当地制作商的色差印成了偏暗的橙色，C店直接使用了"近似的红色"；②三家店的员工制服款式都不相同（A店黑色T恤、B店蓝色polo衫、C店白色衬衫）；③美团/大众点评的头图风格完全不同。品牌部紧急制作了完整的《品牌使用规范手册》（包含设计源文件、色卡实物、排版模板），要求所有新开门店和已有门店的物料更新一律按规范执行。
