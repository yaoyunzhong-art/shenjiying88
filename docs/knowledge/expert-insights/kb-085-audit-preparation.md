# 审计准备清单

> 分类: 财务 | 标签: 审计,内控,财务合规 | 适用: 财务,管理层,门店经理

## 概述

对于连锁电玩城品牌而言，审计可分为两种：外部审计（年度的财务审计、税务专项审计）和内部审计（内控检查、合规性巡查）。无论哪种类型，审计准备工作的质量直接决定了审计的顺利程度和审计意见的好坏。根据行业经验，审计准备充分的公司平均审计周期为3-4周，而准备不充分的公司可能需要8-12周甚至更长——审计人员的加班费和时间成本最终都会被转嫁到审计费用中。

电玩城审计的关键难点在于：收入数据的完整性和准确性（电子支付+现金混合、游戏币消耗的递延收入、团购多渠道结算）、彩票/奖品的库存管理和兑奖负债的计提合理性、以及关联方交易（设备租赁、加盟费、总部管理费分摊）的定价公允性。系统化的审计准备工作应贯穿全年度而非仅审计前一个月突击准备。

## 核心内容

- **全年滚动式准备（非突击式）**: 建立"审计准备月历"——每月固定输出哪些文件（如来款对账单/合同台账/资产变动记录）、每季度固定做什么（如存货盘点/往来款对账/关联交易汇总），这样年度审计时只需要"打包"而不是"从头整理"。具体排期：每月15日前完成上月银行对账；每季度末完成应收账款账龄分析；每半年完成一次全面资产盘点（设备+库存）。

- **核心审计资料清单**: 财务资料（科目余额表、总账/明细账、银行对账单和余额调节表、纳税申报表和完税证明、发票领用存记录、资产清单及折旧计算表、存货盘点表、应收/应付账款明细及账龄分析）；合同资料（设备采购合同/租赁合同/特许经营协议/装修合同/供应商协议/劳动合同）；内控资料（公司的内控制度文件——资金管理/采购管理/收入确认/发票管理等、内控自评报告）；公司基础资料（营业执照/许可证/税务登记/银行开户许可证/股权结构/关联方清单）。

- **收入循环审计重点**: 收入完整性和准确性是审计重中之重。审计师会重点关注：POS系统收入数据 vs 银行收款数据 vs 税务申报数据的三方核对，差异率应<1%；团购平台对账单VS门店确认收入的差异；会员储值收入的递延确认方法是否符合会计准则；彩票兑换的预计负债计提是否充分（基于历史兑换率）；所有的优惠/折扣/赠券的会计处理。建议提前准备"收入穿行测试文件包"——从销售发生到资金入账到收入确认的完整流程截图和凭证。

- **资产与负债的实质性程序**: 固定资产（设备）——盘点清单 vs 财务台账的一致性、新增采购的审批流程和合同对应、固定资产减值测试（是否有闲置/淘汰设备、提前报废审批）；存货——彩票票纸库存、待兑奖品库存的盘点表和成本核算方法（先进先出/加权平均）；银行借款及应付款——融资性租赁的设备对应长期应付款的还款计划表、应付账款的账龄分析和现金支付计划。所有重大资产减值测试需附支持性文档。

- **审计应对策略与注意事项**: ①指定审计对接人（财务+运营各1名），作为审计师唯一的沟通窗口；②审计期间每日召开15分钟总结会（了解当天的审计发现、沟通疑问、安排次日工作）；③对于所有审计师提出的问题，在24小时内响应（即使暂时无法答复也要告知预计答复时间）；④"有诈不等于有错"——审计师提出的质疑和调整建议是正常的审计程序，不要产生对抗情绪；⑤所有审计底稿至少保存6年（法定会计档案保存期限）。

## 实际案例（基于电玩城场景）

成都某电玩城品牌在首次接受外部审计时，由于缺乏审计准备经验，审计师进场后才发现：①设备采购合同丢失2份（约35万元资产无法核对）；②银行余额调节表连续3个月未编制；③彩票库存差异率8%无法解释。审计工作因此被迫延长了3周，额外审计费用约5万元，审计意见也被加了"强调事项段"。此后，门店建立了"财务档案标准化管理制度"并推行月度自审，即使在次年审计时只用了2周完成，审计意见为"无保留意见"。

## 可落地清单

- [ ] 编写《年度审计准备计划》，列出每月/每季度需要准备的审计资料和完成标准
- [ ] 建立"三流一致"核对机制（POS交易流—银行资金流—财务入账），每月核对一次
- [ ] 审计进场前2周进行内部预审，确保所有核心文件齐全、账实相符
