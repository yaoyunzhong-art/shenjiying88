# 新店开业清单

> 分类: 门店管理 | 标签: 开店流程,开业筹备,项目管理 | 适用: 运营团队,项目管理,新店拓展

## 概述

电玩城新店开业是一项复杂的系统工程，涉及选址评估、装修设计、设备采购安装、证照办理、人员招聘培训、营销预热等多个环节。一个标准500-800㎡的电玩城从选址到开业通常需要60-90天，任何一个环节的延误都可能影响整体开业时间，甚至导致前期投入的浪费。据行业统计，开业准备不充分的门店，首月营收低于预期的概率高达35%。

系统化的开业清单管理是确保项目按时按质完成的关键。以shenjiying88的连锁扩张策略为例，建立标准化的开业SOP不仅能够缩短单店筹备周期（目标从75天压缩至55天），更能确保不同城市、不同团队执行的一致性，降低因管理半径扩大带来的质量风险。

## 核心内容

- **选址与合同阶段（D-90至D-75）**: 完成商圈评估（日均人流量≥10,000人次、周边3公里竞品≤3家）、物业条件核查（层高≥3.5m、承重≥500kg/㎡、消防达标）、租赁合同谈判（免租期≥45天、优先续租权）。需输出《选址评估报告》和《租赁合同审核清单》。

- **设计与装修阶段（D-75至D-30）**: 确定平面布局方案（游戏区/休息区/兑换区动线优化）、办理装修报建手续、招标施工团队（要求有娱乐场所装修经验）、制定消防改造方案。需输出《施工进度甘特图》和《消防验收预检表》。

- **设备采购与安装（D-60至D-15）**: 完成设备选型清单（根据客群定位选择街机/游艺/VR/彩票机配比）、签订设备采购合同（含安装调试条款）、设备到货验收、安装调试。需输出《设备布局图》和《设备验收标准》。

- **证照办理（D-60至D-10）**: 营业执照（3-5个工作日）、娱乐经营许可证（15-20个工作日）、消防验收合格证（10-15个工作日）、食品经营许可证（如含餐饮）。建议委托专业代办机构以缩短周期。

- **人员招聘与培训（D-45至D-5）**: 招聘店长（D-60）、收银员/服务员（D-45）、技术人员（D-40）、运营经理（D-50）。完成岗前培训（企业文化、服务标准、设备操作、安全须知），考核通过方可上岗。

- **营销预热（D-30至D-1）**: 确定开业活动方案（充值优惠/免费体验/打卡分享有礼）、完成线上渠道搭建（美团/大众点评/抖音/小红书团购、店铺页面装修）、线下物料制作（招牌/海报/传单/地贴）、KOL探店预约、周边社区宣传。

## 实际案例（基于电玩城场景）

某连锁电玩城品牌在杭州开设第二家门店时，因忽略了娱乐经营许可证的办理周期（实际耗时22天），导致装修完成后空等15天才能开业，日均损失预估营收约3.5万元。后续复盘后将证照办理前置到D-60启动，并采用"双通道并行"策略——装修与证照同时推进，将总筹备周期控制在58天。

## 可落地清单

- [ ] 制定《新店开业里程碑计划表》，明确每个节点的负责人、完成时间和验收标准
- [ ] 每周召开两次跨部门进度会（线上15分钟站会），使用飞书/钉钉项目管理模板跟踪
- [ ] 开业前3天进行全员模拟演练，从迎宾、导玩、兑换到收银全流程跑通
