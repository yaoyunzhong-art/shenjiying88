# NPS净推荐值基准

> 分类: 业务运营 | 标签: NPS,净推荐值,客户忠诚度 | 适用: 管理层,品牌部,门店运营

## 概述

净推荐值（Net Promoter Score, NPS）是衡量客户忠诚度和口碑传播意愿的行业标准指标。NPS的核心问题极简但有力："您有多大可能向朋友/同事推荐本店？（0-10分）"。基于分数将客户分为三类：推荐者（9-10分，乐于向他人推荐品牌）、被动者（7-8分，满意但不会主动推荐）、贬损者（0-6分，对品牌不满甚至可能负面传播）。NPS的最终得分=推荐者占比-贬损者占比（范围-100到+100）。

对于电玩城行业而言，NPS是一个比"满意度"更敏感的指标——它直接关联客户的主动口碑行为。一个NPS得分为+50的门店（推荐者60%、贬损者10%），意味着每10位顾客中就有6位愿意免费为品牌进行口碑传播，而10位中只有1位可能进行负面传播。据行业参考，娱乐休闲行业的平均NPS区间为+20至+40，优秀门店可达+50以上。NPS每提升7个点，企业增长率可提升约1%。

## 核心内容

- **NPS调查实施方法**: 渠道选择：①交易后即时NPS调查——顾客游戏消费完后通过微信小程序/支付页面弹出（最佳时机、回收率高、数据相对真实）；②定期邮件/短信NPS调研（每月1次，发给上次到店≥7天的会员客户）；③第三方平台评价作为"被动NPS"（美团/大众点评5星对应"推荐者"、1-2星对应"贬损者"，但注意样本偏差——只有强烈情绪的人才会主动评价）。调查中除核心问题外，建议追加一个开放问题——“为什么给出这个分数？”——用户的文字反馈才是最宝贵的改进信息。

- **行业基准与目标设定**: 了解行业基准有助于设定合理的NPS目标。通用行业NPS基准：娱乐休闲行业平均+28（来源：Satmetrix 2024行业报告）、连锁实体店平均+25、餐饮行业平均+22、零售行业平均+30。电玩城的典型NPS分布：头部品牌门店+45至+60，中等门店+15至+35，待改进门店-10至+10。建议设定门店年度目标：第一阶段从当前水平提升10个点、第二年再提升10个点、优秀门店目标设定为+50。新店首年的NPS参照同品牌成熟店的标准放宽15-20个点（NPS在开业后3-6个月逐步稳定）。

- **NPS驱动因素分析**: 找出"哪些因素最有可能把顾客从'被动者'变为'推荐者'"。基于调研数据和用户文本反馈，可进行要素关联分析。典型的电玩城NPS驱动因素排序：①设备多样性（是否有足够多、足够新的设备供选择）→②服务态度（员工是否热情、专业、有人情味）→③奖品吸引力（奖品的丰富度和兑换的公平性）→④性价比（是否物有所值）→⑤环境整洁度→⑥等待时间/排队体验。值得注意的是，让顾客成为"贬损者"的原因往往是某个极端的负面体验（设备故障不处理、员工态度恶劣、儿童受伤等）——"一票否决"效应。消除"愤怒因子"比增加"满意因子"更能提升NPS。

- **NPS闭环管理（Close the Loop）**: NPS调查不能"只收数据不回应"。"Closed-loop feedback"（闭环反馈）是NPS体系的核心实践。具体做法：①贬损者（分数0-6分）——收到反馈后24小时内由店长/值班经理电话回访或微信联系，了解不满的具体原因，表达歉意和解决方案。即使只为表达"我们听到了你的声音"，也能挽回不少顾客的心。②被动者（7-8分）——收到反馈后3天内推送感谢信息+轻度优惠（"感谢你的宝贵意见，下次来玩送您5枚币"），引导他们向"推荐者"转化。③推荐者（9-10分）——定期（每季度一次）推送"感谢信"+"老客户专属活动"邀请，或者邀请加入"品牌大使"计划。推荐者是最好的口碑资产。

- **长期NPS趋势分析**: 单月的NPS波动可能是噪声，长期趋势才是有价值的信号。建议关注：月度NPS趋势线（上升/下降/平稳）、六个季度的滚动平均、NPS的季节性趋势（寒暑假/节假日NPS一般更高因为顾客心情好、开学季/工作日的NPS可能有轻微下降）。NPS突然下降时排查：是否设备大面积故障？是否换了一批新员工（磨合期）？是不是某个监管/负面新闻影响了情绪？是否有竞品的重大促销活动导致对比落差感？上升时同理——识别"做对了什么"并放大推广。

## 实际案例（基于电玩城场景）

福州某电玩城2024年Q1 NPS仅有+8，远低于品牌平均+30。深度分析发现贬损者用户在文本反馈中的高频词为"设备老旧""修太慢""态度差"。门店启动了"设备焕新计划"（替换5台老旧机台+加强巡检维护）+"服务态度月"专项（培训+神秘客户暗访+"微笑服务之星"评选）。3个月后NPS稳步提升至+35，Q4稳定在+42。同期月度营收增长约15%。数据和营收的关联证明了——NPS不仅是"调研数据"，更是经营指标。

## 可落地清单

- [ ] 每月收集NPS数据（目标回收量≥门店月度客流的3%），自动化计算和分析
- [ ] 建立"贬损者24小时回访"机制，由店长/值班经理直接处理（不仅是客服）
- [ ] 每季度输出NPS专题分析报告，识别驱动因素的变化趋势和关键改进方向
