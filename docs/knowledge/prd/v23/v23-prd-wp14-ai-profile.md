# V23 WP-14 PRD: C端AI画像与营销引擎

## 概述
基于用户消费行为+活动数据+互动数据，生成多维画像，驱动个性化营销时机推荐、内容推荐、活动策划及周报自动化。

## 需求拆解

### BS-0189: 用户画像聚合 (P1)
- 画像维：性别/年龄/层级(新客/普客/VIP/VVIP)/消费习惯(价值/体验/高端/社交)/兴趣标签
- 消费指标：总额/均价/月消费/最近到店天数/频率/偏好品类/优惠券使用
- 活动指标：总次数/停留时长/偏好时段/偏好日期/高峰到店率
- 互动指标：推送打开率/点击率/消息阅读率/社交分享/评价/评分

### BS-0190: 营销时机推荐 (P1)
- 基于画像计算最佳推送时段/日期/渠道
- 置信度评分
- 个人偏好推导：偏好周末/晚上/免打扰/接受频次

### BS-0191: 内容推荐 (P1)
- 基于兴趣标签生成个性化内容（优惠券/活动/信息/社交）
- 预测打开率和转化率
- 支持限时失效

### BS-0192: 营销活动策划 (P1)
- 创建/启动/完成/取消全生命周期
- 目标群体/时机/渠道/内容关联
- ROI追踪

### BS-0193: 周报 (P1)
- 每周自动生成：用户统计/活动统计
- TOP表现群体
- 运营建议

## API端点
- 17个端点：POST/GET profiles, timing, recommendations, campaigns, reports
- 路由前缀：`/api/ai-profile`

## 无外部依赖
- 纯M5内部能力，不依赖LYT
- 内存存储（生产需接数据库）

## 验收标准
1. 画像创建与查询可正常返回
2. 时机推荐基于画像数据域合理
3. 内容推荐按兴趣标签匹配
4. 活动全生命周期管理可用
5. 周报可生成含正确统计
