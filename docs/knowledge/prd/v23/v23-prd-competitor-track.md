# v23 PRD: Competitor Track (竞品跟踪看板)

## 概述
admin-web 竞品跟踪页面，提供品牌竞品监控看板，支撑市场决策。

## 功能需求

### 数据展示
- 竞品列表表格：品牌名称、城市、综合评分、价格区间、抖音热度、热度趋势
- 统计卡片：跟踪竞品总数、覆盖城市数、高评分品牌数、抖音高热品牌数
- 详情弹窗：品牌介绍、基本信息网格(城市/品类/评分/价格/热度/趋势/门店数/主要区域)

### 筛选与搜索
- 按城市下拉筛选（all + 10个城市）
- 按评分等级筛选（高 ≥4.0 / 中 3.0-3.9 / 低 <3.0）
- 按关键词搜索品牌名称/城市/品类/描述

### 三态
- 加载态：旋转图标 + "正在加载竞品数据..."
- 错误态：错误消息 + 重试按钮
- 空态：无数据提示 + 清除筛选

### 分页
- 每页5条，分页导航（首页/末页/上一页/下一页）

## 种子数据
10条竞品记录：瑞幸咖啡、星巴克、Manner Coffee、霸王茶姬、蜜雪冰城、喜茶、奈雪的茶、幸运咖、一点点、库迪咖啡

## 技术方案
- `'use client'` 组件，内联样式
- 类型定义 `CompetitorRecord` / `ScoreLevel`
- 辅助函数 `filterCompetitors / getScoreLevel / formatPrice`
- 导出供测试使用
