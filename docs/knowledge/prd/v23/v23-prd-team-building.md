**产出**: `apps/admin-web/app/team-building/page.tsx` + `.test.tsx`
**作用**: V23 团队建设管理核心功能

## 1. 概述

团建活动管理（Team Building Management）面向公司管理员，提供团建活动全生命周期管理能力。管理员可以查看所有活动的概览统计，浏览活动列表，新建团建活动，并按活动状态进行筛选。

## 2. 功能范围

### 2.1 活动概览统计
| 指标 | 说明 |
|------|------|
| 总活动数 | 全部团建活动数量 |
| 进行中 | 当前进行中的活动数量 |
| 已完成 | 已完成的活动总数 |
| 总参与人次 | 所有活动累计参与人次 |

### 2.2 活动列表
| 特性 | 说明 |
|------|------|
| 列表展示 | 活动名称/日期/状态/参与人数/预算等字段 |
| 状态筛选 | 全部/待报名/进行中/已完成/已取消 |
| 搜索 | 按活动名称搜索 |
| 排序 | 按时间倒序排列 |

### 2.3 新建活动
| 字段 | 说明 |
|------|------|
| 活动名称 | 必填，1-50字 |
| 活动日期 | 必填，日期选择 |
| 活动地点 | 必填 |
| 预算 | 必填，正数 |
| 预计参与人数 | 必填，正整数 |
| 活动描述 | 选填 |
| 活动类型 | 团队拓展/聚餐/旅游/运动/其他 |

### 2.4 三态覆盖
| 状态 | 渲染内容 |
|------|----------|
| 加载中 | 加载动画 |
| 错误态 | 错误提示+重试按钮 |
| 空态 | 无数据提示+新建引导 |

## 3. 技术方案

- 前端框架: Next.js App Router + React 18
- UI 组件: @m5/ui (StatCard / DataTable / Modal / SearchFilterInput / Button)
- 三态: loading / error / empty
- 弹窗: 内联 Modal 组件实现新建表单
- 默认样本: 10 条种子数据
