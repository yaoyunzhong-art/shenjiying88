# PRD-009: 联名券 — Co-branded Coupon (P-48)

> 版本: v1.0 · 签发人: 🦞 龙虾哥 · 对接专家: E33 联名
> 发布日期: 2026-07-13 · 状态: 🟢 已签发
> 关联Phase: P-48

## 1. 业务背景

店A与其他品牌（奶茶店/电影院/健身房）联名发券，互相导流。当前无联名券系统。

## 2. 功能需求

| RQ | 标题 | 优先级 | 验收标准 |
|:---|:-----|:------:|:---------|
| RQ-48-01 | 联名券创建 | P0 | 选择合作品牌→设置优惠规则→创建 |
| RQ-48-02 | 联名券发放 | P0 | 用户到店消费→自动发放联名券 |
| RQ-48-03 | 联名券核销 | P0 | 用户在合作店出示→核销使用 |
| RQ-48-04 | 联名券查询 | P1 | 用户查看我的卡包→联名券列表 |

## 3. 验收卡

| AC | 场景 | 预期 |
|:---|:-----|:-----|
| AC-48-01 | 创建联名券(满100减20) | 联名券创建成功 |
| AC-48-02 | 消费后自动发券 | 完成订单→用户获得联名券 |
| AC-48-03 | 核销联名券 | 核销成功→券状态=已使用 |
| AC-48-04 | 过期联名券 | 有效期已过→券不可用 |

## 4. 数据模型

```typescript
interface CoBrandedCoupon {
  id: string;
  brandName: string;        // 合作品牌
  type: 'discount' | 'cash' | 'gift';
  value: number;
  condition: string;        // 使用条件
  validFrom: Date;
  validTo: Date;
  memberId: string;
  status: 'active' | 'used' | 'expired';
}
```
