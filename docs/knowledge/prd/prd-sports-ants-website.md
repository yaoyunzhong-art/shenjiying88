# 运动蚂蚁（BigAnts）品牌官网 PRD

## 1. 项目概述

### 1.1 项目名称
运动蚂蚁（BigAnts）品牌官网

### 1.2 项目类型
企业品牌展示型网站（B2B）

### 1.3 项目定位
- 品牌官网：展示运动蚂蚁品牌形象、公司实力、产品体系
- 营销转化：获取潜在客户咨询、项目合作、加盟代理
- 行业标杆：数字运动潮玩行业领导者形象

### 1.4 目标用户
- 投资商/加盟商：寻求数字运动场馆项目合作
- 商业地产商：购物中心、游乐园、文旅项目
- 政府采购/企业采购：单位娱乐设施采购
- 渠道商/代理商：区域代理合作

## 2. 品牌信息

### 2.1 品牌基本信息
- **品牌名称**：运动蚂蚁（BigAnts / Sports Ant）
- **母公司**：广州蚂蚁运动有限公司
- **品牌定位**：数字潮玩运动一站式提供商
- **品牌口号**：带你玩转数字体育

### 2.2 核心业务
1. 数字运动设备研发、生产、销售、服务
2. 数字运动馆规划、设计、施工、运营
3. 品牌直营、合作、加盟

### 2.3 产品体系
1. **模拟运动系列**：网球模拟器、棒球模拟器、高尔夫模拟器、守门员模拟器
2. **射击系列**：枪王之王、奥运射击项目
3. **VR/AR系列**：VR滑雪、AR寻宝、激光迷宫
4. **大型游戏设备**：飞行影院、极限漂流、动感影院
5. **娱乐设备**：拳击游戏机、唱歌机、超级滚球
6. **互动游戏**：超级方块、激光矩阵

### 2.4 品牌数据
- 2000+ 合作客户
- 50+ 国家和地区
- 500+ 场地案例
- 100+ 专利认证
- 60+ 款数字运动类设备

### 2.5 品牌理念
- **公司使命**：让体育智慧变得有趣
- **公司愿景**：让运动蚂蚁成为数字运动领域的领导者，甚至是行业的标杆
- **核心价值观**：客户成功，团队成功，员工成功
- **服务理念**：通过服务创造价值，为客户提供最合适的解决方案

## 5. 视觉设计规范

### 5.1 色彩体系
运动蚂蚁品牌色彩以科技感、运动感为主：
- **主色调**：科技蓝 #0066FF
- **辅助色**：活力橙 #FF6B00
- **强调色**：智慧绿 #00C853
- **背景色**：深空灰 #1A1A2E、纯白 #FFFFFF
- **文字色**：主文字 #1A1A2E、次文字 #666666

### 5.2 字体规范
- **中文**：思源黑体（Source Han Sans）/ 阿里巴巴普惠体
- **英文**：Inter / Roboto
- **数字**：DIN Alternate / Roboto Mono

### 5.3 视觉风格
- 科技感：渐变背景、粒子效果、光效元素
- 运动感：动感线条、速度元素、力量表现
- 现代感：大尺寸图片、简洁布局、呼吸感留白
- 品牌元素：蚂蚁形象、运动轨迹、科技网格

### 5.4 动效规范
- 页面切换：淡入淡出 300ms ease-out
- 元素进入：上移渐显 400ms ease-out，间隔 100ms
- 悬停效果：缩放 1.02，轻阴影 200ms
- 滚动视差：背景层 0.5x，内容层 1x
