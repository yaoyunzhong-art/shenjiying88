# 交易链 Smoke 验收模板

> 用途: 在每轮交付前快速确认主链可演示、可验收、状态正确
> 适用场景: storefront-web 收银/结算/支付/退款/财务

## 前置条件

- 本地后端已启动（`pnpm dev`）
- storefront-web 已启动（`pnpm dev`）
- 浏览器可访问 `localhost:3000/cashier`
- 种子数据已加载（环境 `NODE_ENV !== 'production'`）

---

## Smoketest 清单

### 1. 收银（cashier）

- [ ] **商品加载**：页面打开后可见商品列表，非空
- [ ] **商品搜索**：输入关键字，结果实时过滤
- [ ] **添加/修改/删除购物车**：左侧 + 按钮 → 已选清单更新
- [ ] **会员识别**：输入 11 位手机号 → 查询 → 显示等级/积分/折扣
- [ ] **支付方式选择**：可选 微信/余额/现金，选中后高亮
- [ ] **空结算防御**：购物车为空时点击结算 → 提示
- [ ] **结账流程**：选商品 + 会员 + 支付方式 → 结算 → 跳转 H5 支付页

### 2. 结算（checkout）

- [ ] **商品列表**：页面商品来自后端，非 hardcode
- [ ] **金额计算**：小计、配送费、合计展示正确
- [ ] **优惠券验证**：输入有效券码 → 显示优惠减免；无效券码 → 显示错误
- [ ] **表单验证**：必填字段缺失时禁止提交
- [ ] **提交订单**：填写完整 → 提交 → 跳转 H5 支付页

### 3. 订单（orders）

- [ ] **订单列表**：有数据时展示分页列表；无数据时展示空态
- [ ] **筛选**：按状态（待支付/已支付/已退款）筛选正确
- [ ] **搜索**：按订单号搜索定位
- [ ] **订单详情**：点击订单 → 展示聚合信息（商品清单、金额、支付状态）
- [ ] **退款发起**：已支付订单可发起退款（弹窗 + 金额/原因输入）
- [ ] **退款记录**：退款后详情页可见退款记录

### 4. H5 支付

- [ ] **支付页**：订单号、金额正确展示
- [ ] **二维码**：`qrCode` 存在时渲染 `QRCodeDisplay`
- [ ] **轮询刷新**：5s 自动轮询订单状态并更新
- [ ] **倒计时**：待支付状态展示二维码过期倒计时
- [ ] **支付结果页**：支付成功后自动刷新 → 展示成功画面
- [ ] **失败/超时态**：支付失败或二维码过期时展示正确提示

### 5. 财务（finance）

- [ ] **总览卡片**：总营收、退款、净收入、流水笔数正常
- [ ] **收入趋势**：月度趋势图/数据正确
- [ ] **流水记录**：明细列表正确，可按类型/日期筛选

### 6. 会员权益

- [ ] **会员登录**：可识别会员（手机号查询）
- [ ] **权益展示**：checkout 页右侧可见余额/积分/优惠券数
- [ ] **会员支付**：余额支付选项可用

---

## 验收标准

| 等级 | 要求 |
|------|------|
| **PASS** | 全部用例通过，无阻断性缺陷 |
| **CONDITIONAL PASS** | 非核心用例失败但已确认可接受原因 |
| **FAIL** | 存在阻断主链的缺陷 |
