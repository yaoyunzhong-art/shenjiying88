## 四、操作指引卡片（预印发放）

### 收银员操作卡（15 人）

```
【第 1 步】打开浏览器 → 访问收银台
【第 2 步】确认商品列表加载完成（应有 10+ 商品）
【第 3 步】点击商品加入购物车（选 2~3 件）
【第 4 步】输入会员手机号 → 确认姓名和折扣
【第 5 步】选择支付方式 → 点击"结算"
【第 6 步】跳转到 H5 支付页 → 看到二维码/支付链接
【第 7 步】返回订单列表 → 确认订单存在
【第 8 步】记录操作感受：________
```

### 店长操作卡（16 人）

```
【第 1 步】登录 → 进入会员中心
【第 2 步】搜索会员 → 查看余额/积分/等级
【第 3 步】进入订单管理 → 按状态筛选
【第 4 步】进入财务总览 → 查看营收趋势
【第 5 步】体验员工排班/库存预警页
【第 6 步】记录操作感受：________
```

### 客服操作卡（10 人）

```
【第 1 步】打开订单列表 → 搜索特定订单号
【第 2 步】点击订单进入详情 → 查看商品/金额/支付状态
【第 3 步】"发起退款"→ 输入金额 + 原因 → 提交
【第 4 步】确认订单状态 → 已变为"退款中"
【第 5 步】反复操作 10 次（不同订单）
【第 6 步】记录操作感受：________
```
