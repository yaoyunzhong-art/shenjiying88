1. **L1/L2从未被自动化** —— 白皮书写了"追加到验收脉冲"，但脉冲代码本身从未修改。没有任何人（龙虾哥/树哥）去改cron或脉冲脚本。

## 二、原始V21方案（v1，未更改）

### V21 四层持续进化引擎

```
┌─────────────────────────────────────────────────────────┐
│               🔬 四层持续进化引擎                          │
├─────────────────────────────────────────────────────────┤
│  L1 实时反循环 (每次10min脉冲触发)                          │
│  L2 每次修复后闭环 (树哥+龙虾哥回写)                        │
│  L3 每日23:00趋势总结 (替代旧checklist)                     │
│  L4 每周日进化批处理 (全量数据跑分+策略调优)                  │
└─────────────────────────────────────────────────────────┘
```

### L1: 实时反循环（每次验收脉冲附带）

**触发**: 每次验收脉冲末尾自动执行
**输入**: 本次脉冲发现的所有异常（测试fail、TSC错误、树哥违规）
**处理**:
1. 异常分类（类型+模块+根因标签）
2. 查反模式库是否有匹配（已有的直接引用，新增的临时记录）
3. 如果同一模块连续3次脉冲同类型fail → 自动升级到L3
**产出**: `memory/evolution/snapshot-YYYY-MM-DD-HHMM.md`

### L2: 每次修复后闭环回写

**触发**: 树哥或龙虾哥修复一个bug/违规后
**处理**:
1. 修复的bug标题和内容
2. 根因分析(1-2句)
3. 可重复执行性评分(1-5)
4. 如果要记入反模式/正向模式库，标注"+AM"或"+PP"
**产出**: 只更新 evolution-log.md，不额外创建文件

### L3: 每日23:00趋势总结（替代旧checklist）

**输入**: L1产生的所有snapshot + L2修复记录
**产出**:
1. **当日健康评分** (0-100):
   - 测试通过率 = 当日测试pass / (pass+fail) × 40分
   - 树哥违规率 = (1 - 违规模块/总交付模块) × 20分
   - 代码质量 = (1 - TSCfail/总文件) × 20分
   - 闭环率 = 当日修复完成/当日发现异常 × 20分
2. **趋势变化**: 与昨日相比三项核心指标↑↓
3. **反模式动效**: 哪些反模式今天出现/消失/重复
4. **V(N+1)计划改进**: 基于数据的建议
5. **自动产出**: `docs/knowledge/daily-brief.md` 追加进化摘要

### L4: 每周日进化批处理

**触发**: 每周日 23:00自动
**输入**: 过去7天的L3产出
**处理**:
1. 全量数据跑分（7天平均值、峰值、谷值）
2. 反模式频率统计（哪些反模式出现最多？）
3. 树哥各agent的个体评分（合规率/交付质量）
4. 策略调优建议：
   - 哪些反模式需要新增圈梁箍
   - 哪些正向模式可以固化到MEMORY.md
   - 派单模板是否需要改进
**产出**: `docs/knowledge/evolution-weekly.md`

## 三、V21 健康评分计算公式（v1保留）

### 日评分 (0-100)

```
日评分 = 
  测试通过分(40) + 
  树哥合规分(20) + 
  TSC稳定分(20) + 
  闭环率分(20)

测试通过分 = min(40, (当日通过测试数 / 最大预期每日测试数 × 40))
树哥合规分 = 20 × (1 - 违规次数 / max(1, 交付模块数))
TSC稳定分 = 20 × (1 - TSC错误文件数 / 总文件数)
闭环率分 = 20 × (修复模块数 / max(1, 发现异常的模块数))
```

### 周评分 (0-100)

```
周评分 = 
  日评分均值(60) + 
  趋势奖励/惩罚(20) + 
  新模式贡献(20)

趋势奖励 = 如果周中健康评分呈上升趋势 → +10～20
趋势惩罚 = 如果周中健康评分呈下降趋势 → -10～-20
新模式贡献 = 新增正向模式数 × 5 - 新增反模式数 × 5
```

### 评分阈值
| 评级 | 范围 | 含义 | 行动 |
|:----:|:----:|:-----|:-----|
| 🟢 S | 95-100 | 优秀 | 维持 |
| 🟢 A | 80-94 | 良好 | 微调 |
| 🟡 B | 60-79 | 一般 | 需改进 |
| 🟠 C | 40-59 | 差 | 干预 |
| 🔴 D | 0-39 | 临界 | 立即检查 |
