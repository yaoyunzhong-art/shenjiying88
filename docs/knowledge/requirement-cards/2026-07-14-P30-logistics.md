# P-30 SSE后勤 · 需求卡片
> 创建: 2026-07-14

## PRD 收口
- PRD-010: SSE后勤管理 — Logistics (P-30)

## 业务需求
P-30 的核心是“设备巡检 + 清洁排班 + 维修工单 + 物料申领”。当前仓库已经把四条主链都做成了真闭环：`apps/api/src/modules/logistics/` 既提供巡检任务创建、提醒发送、结果记录与“已巡检”状态回写，也提供清洁排班的 `排班 -> 区域分配 -> 考勤签到` 主链、维修工单的 `报修 -> 派单 -> 维修 -> 验收` 状态链，以及物料申领的 `申领 -> 审批 -> 出库` 主链；`apps/admin-web` 的巡检页、排班页与库存页继续承担前台样板。
