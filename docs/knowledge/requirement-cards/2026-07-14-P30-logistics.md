# P-30 SSE后勤 · 需求卡片
> 创建: 2026-07-14

## PRD 收口
- PRD-010: SSE后勤管理 — Logistics (P-30)

## 业务需求
P-30 的核心是“设备巡检 + 清洁排班 + 维修工单 + 物料申领”。当前仓库已经把 `RQ-30-01` 做成了真闭环：`apps/api/src/modules/logistics/` 提供巡检任务创建、提醒发送、结果记录与“已巡检”状态回写；`apps/admin-web` 的巡检页面继续承担前台样板。其余 `清洁排班 / 维修工单 / 物料申领` 仍然处于缺口状态。

## 当前缺口
- `RQ-30-02` 清洁排班尚无独立模块、排班表、区域分配、考勤逻辑
