# 📋 需求卡片目录

> 专家体系改革V1：报告→任务卡模式
> 每个卡片 = 1个决策点 = 1个具体行动

## 卡片格式
```
📋 卡片ID | 日期
═══════════════════════
模块:     [模块名]
专家:     [E-XXX]
问题:     [一句话问题]
决策:     ✅ 通过 | ❌ 退回 | ⏸️ 待定
行动:     [具体做什么]
负责人:   [谁]
截止:     [什么时间]
═══════════════════════
```

## Gate签名格式
```
📋 Gate#[1-6] · 签署卡
═══════════════════════
门:   Gate 1 架构+安全 · E1陈架构
模块: [模块名]
版本: [commit hash]
结论: ✅ 通过 · ❌ 退回 · ⏸️ 待定
附注: [不超过100字]
时间: 5分钟超时自动通过
═══════════════════════
```
