## PRD 收口
- PRD-012: 品牌运营 — Brand Operations (P-47)

## 业务需求
P-47 的核心是“品牌官网管理 + 品牌活动 + 品牌素材 + 门店品牌同步”。当前仓库并非空白实现，`brand-custom / marketing / content` 三块已经形成主链，只是一直没有被正式收成 `Phase / PRD / 测试 / 审计` 的统一口径。

## 树哥接单清单
- `apps/api/src/modules/content/content.service.ts + content.controller.ts`
  - 对齐 `RQ-47-01 / AC-47-01`
  - 负责官网内容创建、编辑、发布、归档、搜索
- `apps/api/src/modules/marketing/marketing.service.ts + marketing.controller.ts`
  - 对齐 `RQ-47-02 / AC-47-02`
  - 负责品牌活动、A/B、优惠券发放、ROI、归因、触达渠道
- `apps/api/src/modules/brand-custom/brand-custom.service.ts + brand-custom.controller.ts`
  - 对齐 `RQ-47-03 / RQ-47-04 / AC-47-03`
  - 负责主题、logo/domain/email template、多品牌隔离与同步配置
- `apps/api/src/modules/brand-custom/brand-custom.phase-p47.test.ts`
  - 作为 P-47 的 Phase 主证据
