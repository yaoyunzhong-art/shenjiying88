# P-49 SEO/GEO 智能优化 · 需求卡片
> 创建: 2026-07-14 | 截止: 7/30 (P-49 前台流量入口收口)

## PRD 收口
- PRD-015: SEO/GEO 智能优化

## 业务需求
E44 开放平台: 除了租户 AI 接入与开放 API，P-49 还必须打通前台搜索流量与 AI 引用流量入口，确保官网页面具备可抓取、可引用、可地域化、可监控的基础能力。

## 技术方案
E1 陈架构: `apps/tob-web` 作为前台承载，根级 `app/sitemap.ts` 与 `app/robots.ts` 暴露站点入口；`brand-website` 承担品牌官网 SEO/GEO 主实现；`sports-ants` 提供转化追踪与营销页埋点锚点；`lib/seo`、`lib/geo`、`lib/intelligent` 承担元信息、地域化与自治优化能力。

## 树哥接单清单
- `root sitemap/robots`: 对齐 `RQ-49-12` / `AC-49-12` / `AC-49-13`
- `brand-website sitemap/robots/geo`: 对齐 `RQ-49-12` / `RQ-49-14` / `AC-49-15`
- `meta-generator + JSON-LD`: 对齐 `RQ-49-11` / `RQ-49-13` / `AC-49-11` / `AC-49-14`
- `ai-reference-optimizer`: 对齐 `RQ-49-15`
- `intelligent/self-system`: 对齐 `RQ-49-16` / `RQ-49-18`
- `sports-ants + brand-website 页面`: 对齐 `RQ-49-17`

## 优先顺序
1. P0: sitemap / robots / meta / JSON-LD / GEO 路由
2. P1: AI 引用优化与地域化内容输出
3. P1: 监控快照、异常识别与优化任务闭环
4. P2: 浏览器级页面验收与监控看板实地核验
