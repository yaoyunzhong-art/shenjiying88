## 时间线（固定锚点·预算法则）

> **核心规则（每条cron触发时自动判断）：**
> ```
> ⏱ 当前时刻 → 下个锚点的剩余时间 >= 任务预算 → 开跑
> ⏱ 当前时刻 → 下个锚点的剩余时间 < 任务预算  → 不跑，直接等下一个锚点
