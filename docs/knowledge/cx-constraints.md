## 四大核心原则（详见归档层 §1）

| 原则 | 核心约束 |
|:----:|---------|
| **便捷** | 核心操作≤3步; 关键信息一屏展示 |
| **科学** | 100%营销推送有AI决策记录; 推荐匹配度>80% |
| **少打扰** | 每日P2+P3合计≤1条; 23:00-08:00静默; 一键关闭P3 |
| **AI精准** | 转化率提升≥30%; 投诉率下降≥50% |

## 推送分级 (详见归档层 §2)

| 级别 | 示例 | 关闭? |
|:----:|------|:----:|
| P0 交易类 | 消费确认·退款·合同变更 | ❌ |
| P1 服务类 | 预约确认·权益变动·到期提醒 | ❌ |
| P2 权益类 | 月度账单·升级通知·积分变动 | ✅ |
| P3 营销类 | 活动促销·新品推荐·品牌传播 | ✅ |

### 2026年7月合规补充
#### AI生成趣味引导标识要求
- 机台AI得分推荐/推荐理由标注 `🤖 AI分析`
- AI生成的优惠券/活动推荐须标注尾注说明
- 赛事AI排位/匹配结果须提供可点击「查看算法依据」

#### 数据最小必要原则落地
- 采集数据字段清单须经法务逐项确认
- 会员注册: 手机号+昵称即可(不强制实名)
- 设备绑定: 仅需设备MAC+SN, 不关联用户个人信息
- 评论/评价: 仅统计级使用, 不暴露UGC到竞品分析面

### 1. 个保法2026修订要点
根据律师解读, 2026年个保法修订核心要点:
- **用户权利强化**: 扩大知情权范围, 自动化决策须提供"拒绝权"便捷入口
- **合规处罚明确**: 数据分类分级义务法定化, 违规处罚参照《数据安全法》最高¥5,000万
- **跨境数据**: 数据出境安全评估常态化
- **审计要求**: 合规审计由"建议"变为法定义务, 频次根据处理量(100万/1000万分级)
- **对shenjiying88影响**: 需重点关注: ①会员数据最小化收集 ②推送拒绝权 ③合规审计频次 ④未成年人个保

Source: 詹俊宇律师《个人信息保护法2026年修订要点解析》

### 2. GA/T 2380-2026等保数据安全新规对C端的影响
2026年6月1日实施的等保数据安全新规要求:
- 数据全生命周期8环节安全覆盖(收集→销毁)
- 三级(重要数据/≥100万人个信): 存储加密+日志留存≥1年+溯源
- 敏感个人信息脱敏展示(会员手机号/身份证)
- **对C端直接影响**: 用户需感知到数据保护级别的透明度标识
- 推送记录中涉及个人偏好的部分需要加密存储

### 4. 电玩城用户C端新发现(基于全球市场数据)
- 2025年全球电玩城核心用户25-34岁占**47.3%**, 单人单次平均¥98
- 青少年(12-18)约30%, 传统认知偏低(但仍是重要客群)
- 非游戏收入(餐饮/IP衍生品)占营收**29%** → C端体验需要覆盖"从体验到消费"全链路
- 订阅制会员月卡收入占比**22%** → 会员权益设计要更关注高频用户的特权感
- 自助结算+无人值守15%覆盖率 → 自助体验的流畅度(扫码/支付/兑换)直接影响NPS

Source: ZVZO《全球及中国游戏厅娱乐市场趋势深度分析报告》(2026)

### 5. 2026年门店SaaS用户痛点(交叉参考)
- 中小商户选SaaS时最关注: 稳定性>功能适配>服务响应>生态兼容>成本效益
- 连锁商户核心痛点: 多门店统一管控、数据汇总、员工权限
- 中小商户核心痛点: 操作简单、价格亲民、生态开放
- **对C端(店主)的体验约束**: SaaS系统本身需要易上手(20分钟教学), 否则拖累整个生意的数字化

Source: 博客园 2026门店SAAS系统测评

### 6. 2026-07-16 日采新增 — 行业合规与CX约束更新

#### 6.1 电玩城用户行为数据补充
来源: 2026网易深度测评·魔幻之城方案·12家头部走访

- 云平台支持的"游戏行为画像"功能可根据用户偏好推荐个性化任务
- 示例: "同机台高分手榜挑战""好友组队赛"→日均停留时长**120分钟+**
- 用户从扫码到开玩时间压缩至**3秒以内** → CX核心指标: 扫码→开玩≤3s
- OTA推送"限定款皮肤""节日主题关卡" → CX约束: 内容更新≥2次/月
- 电竞化趋势: 排行活动·组队挑战→推送需包含社交元素(而非纯营销)

#### 6.2 PUSH推送频控增强
基于电玩城新发现的用户分组(P1青少年30%/P2情侣35%/亲子25%/中老10%):
- 青少年(12-18): 周末P3可接收(限六日,每日≤2条), 工作日仅P0+P1
- 情侣(19-28): P2上限(每日≤1条), 推荐内容含约会/赛事/盲盒类
- 亲子(30-45): 仅P0+P1(消费确认+兑换提醒+亲子赛事邀约)
- 中老(45+): 仅P0(消费提醒), 其他级别不推送
- ⚠️ 14岁以下: 默认关闭P2/P3, 需家长开启

#### 6.3 SaaS系统易用性约束(对店主层面)
来源: 2026门店SAAS系统测评
- **20分钟教学**是SaaS的CX硬门槛: 超时=低满意度
- 电玩城员工流动性强→系统操作需≤3步直达核心功能
- 中小商户关注点: 操作简单>价格>功能>生态兼容
- CX原则补充: 新手引导流程需≤5分钟完成核心操作教学

#### 6.4 自助/无人值守CX约束
- 自助结算+无人值守覆盖15%存量门店(2025)→2026目标**30%+
- CX关键指标: 扫码→支付→出币(或兑换)全流程**≤10秒**
- 离线场景(地下室/改造): 本地扫码·支付·兑换可用(无需网络)
- 支付失败容错: 重试≤3次, 超时自动退款+短信通知
