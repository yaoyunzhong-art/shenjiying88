## 四大核心原则（详见归档层 §1）

| 原则 | 核心约束 |
|:----:|---------|
| **便捷** | 核心操作≤3步; 关键信息一屏展示 |
| **科学** | 100%营销推送有AI决策记录; 推荐匹配度>80% |
| **少打扰** | 每日P2+P3合计≤1条; 23:00-08:00静默; 一键关闭P3 |
| **AI精准** | 转化率提升≥30%; 投诉率下降≥50% |

## 推送分级 (详见归档层 §2)

| 级别 | 示例 | 关闭? |
|:----:|------|:----:|
| P0 交易类 | 消费确认·退款·合同变更 | ❌ |
| P1 服务类 | 预约确认·权益变动·到期提醒 | ❌ |
| P2 权益类 | 月度账单·升级通知·积分变动 | ✅ |
| P3 营销类 | 活动促销·新品推荐·品牌传播 | ✅ |

## 2026年最新合规补充

### 2026 AI监管法规 (新增)

| 法规 | 影响 |
|------|------|
| 算法推荐管理规定(2022) | 个性化推荐需算法备案 |

| 生成式AI管理办法(2023) | 训练数据合法性审查 |

| 个人信息保护法(2021) | 推送同意+数据最小必要 |

### AIGC内容标识要求（新增）

- AI生成推送内容须标识「本内容由AI生成」
- 推荐理由需可展示算法依据(匹配度/画像快照)
- 推送记录表需增加 `is_ai_generated` 字段

### 频控+opt-out合规增强（新增）

| 原约束 | 增强 |
|--------|------|
| 每日≤1条 | 增加每月≤30条上限 |
| 23:00-08:00静默 | 增加节假日静默(P3) |
| 一键关闭P3 | opt-out记录保留6个月, 关闭后360天不得启用 |
| — | 年度合规复盘报告 |

### 90天合规行动清单

| 天数 | 动作 | 负责 |
|:----:|------|:----:|
| 0-30天 | 盘点AI生成营销场景/数据分级/协议覆盖 | 法务+产品 |
| 31-60天 | 高风险场景加人工复核与留痕 | 技术+运营 |
| 61-90天 | 算法说明/投诉渠道草案/供应商问卷 | 法务+采购 |

### 当前差距（4处待修）

### 2026年7月合规补充

#### AI生成趣味引导标识要求
- 机台AI得分推荐/推荐理由标注 `🤖 AI分析`
- AI生成的优惠券/活动推荐须标注尾注说明
- 赛事AI排位/匹配结果须提供可点击「查看算法依据」

#### 数据最小必要原则落地
- 采集数据字段清单须经法务逐项确认
- 会员注册: 手机号+昵称即可(不强制实名)
- 设备绑定: 仅需设备MAC+SN, 不关联用户个人信息
- 评论/评价: 仅统计级使用, 不暴露UGC到竞品分析面
