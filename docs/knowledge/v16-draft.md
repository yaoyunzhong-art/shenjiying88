### 🔴 突破1：专家体系改革落地（P0）
**从"纸上谈兵"到"实际运行"**

```
改动步骤:
  1. 修改专家cron → 产出决策卡片（非散文报告）
     - 每卡三要素：一句话结论 + 发现的1-3个问题 + 行动卡片ID
     - 每天仅有2位专家轮值写报告
  
  2. 报告引入6道Gate签名（手动确认）
     - Gate自动票 → 推送到龙虾哥 → 确认 or 退回
     - 超时5分钟自动通过（不阻塞开发）
  
  3. 周2次代码抽查（周三+周五）
     - E1从最新commit随机抽3个TS文件打分 A/B/C/D
     - C=退修, D=整模块重写
     - 结果录入phase-progress
```
