## 3️⃣ 54专家参与审查系统：每天执行（V2）

> 详见 `docs/knowledge/expert-participation-solution.md`

### 核心规则

| 规则 | 内容 |
|:-----|:------|
| 频率 | **每天8人轮换，7天全覆盖54人** |
| 时间 | 每人每次**5分钟** |
| 启动 | **龙虾哥直接点名，不等人自荐** |
| 门槛 | 读3个文件 / 跑1个场景 / 点1个按钮 |
| 后果 | 当天未签→**日报曝光→晨会追问** |
| 换人 | 连续3天不签→**移出专家名单** |

### 审查不占用开发时间——嵌入在间隙里

审查任务发出去后，专家回复不需要龙虾哥盯。5分钟后看一眼回复就行。
