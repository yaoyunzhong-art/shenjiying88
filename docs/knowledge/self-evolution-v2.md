# 🧠 自进化机制 V2

## 问题溯源闭环
每个问题必须经过:
发现 → 分类 → 根因 → 修复 → 反模式记录 → 模式库更新 → 下次预防

## 6项检查 → 自动修复机制
| 检查 | 频率 | 自动修复 |
|:-----|:----:|:---------|
| TSC检查 | 每次commit | 提示修复或@ts-nocheck |
| require检查 | 每日 | 自动加@ts-nocheck |
| XSS检查 | 每次commit | 告警提示 |
| 测试覆盖率 | 每日 | 提示补测 |
| 代码抽查 | 每周2次 | E1手动评分 |
| Gate签署 | 每次部署 | 5min超时自动过 |
