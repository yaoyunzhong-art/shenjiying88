## 💡 投放策略矩阵

### 淡季 (周一~周四)
```
策略: 低成本获取流量 + 私域运营
预算: 正常60%
核心动作:
  - Google Ads: 用低出价覆盖搜索流量
  - Meta: 针对学生人群推weekday半价
  - TikTok: 正常内容发布，不投广告
  - 私域: 唤醒沉睡客户、VIP推送
```

### 旺季 (周五~周日)
```
策略: 全渠道收割
预算: 正常150%
核心动作:
  - Google Ads: 提高出价，抢占周末搜索流量
  - Meta: 增大预算，推周末活动素材
  - TikTok: 投Spark Ads + 周末特惠视频
  - GBP: 周末活动Post
  - 私域: 周四/五分批发周末活动邮件
```

### 季节/节日脉冲
```
策略: 短期高强度投放
预算: 2-3倍（提前1-2周开始）
核心动作:
  - 提前2周: 制作节日主题内容
  - 提前1周: 开始投放节日广告素材
  - 提前3天: 私域推送节日活动
  - 节日当天: 全渠道同步内容
  - 节后3天: 复盘+再利用素材
```
