# 🛠️ USA Street Game MCP — 配置目录

> shenjiying88 体系 · 美国街机 MCP 子模块

## 项目简介

本目录是 **usa-street-game-mcp** 项目的配置子模块，存放 MCP（Model Context Protocol）街机游戏的后端配置、营收模型参数及运营策略模板。面向 **shenjiying88** 体系下美国市场的单店长一人运营战斗模型，支撑月预算 $2,000–$5,000 的街机业务精细化运营。

## 目录结构

```
usa-street-game-mcp/configs/
├── README.md              ← 🟢 本文件（配置目录说明）
└── revenue-model.md       ← 💵 营收模型与预算分配指南
```

## 快速开始

1. **查看营收模型**：打开 `revenue-model.md` 获取完整的营收分配规则、预算管理策略和利润计算方式
2. **扩展配置**：如需添加新的配置文件（游戏定价策略、地域参数、设备分组等），请在 `configs/` 下按 `.md` 或 `.json` 格式创建新文件
3. **与 MCP 服务集成**：所有配置通过 MCP 协议被 AI 代理调用，确保配置格式与协议兼容

## 配置说明

| 文件 | 类型 | 说明 |
|------|------|------|
| `revenue-model.md` | 营收模型 | 单店月预算分配、收入分成比例、成本控制基线 |

- 所有配置遵循 **一人运营** 模式，兼顾效率与灵活性
- 配置变更需同步更新 `usa-street-game-mcp/README.md` 中的索引

## API / 模块说明

本目录不直接暴露 API，其配置内容由 `usa-street-game-mcp` 模块通过 MCP 协议向 AI 代理暴露。配置数据作为上下文注入到街机游戏相关的 prompt 中，指导营收决策和运营策略推荐。

## P-38 财务冲刺

**USA Street Game MCP 配置模块**在 **P-38 财务冲刺** 中承担以下角色：

- **营收模型基准**：提供单店运营的收支基线，作为 P-38 财务对账与损益分析的输入依据
- **预算参数源**：`revenue-model.md` 中定义的月预算分配比（$2,000–$5,000）直接服务于 P-38 的现金流模拟与 ROI 评估
- **成本控制基线**：一人运营模式下的成本结构，为 P-38 降本增效策略提供数据支撑
- 与 `docs/acceptance/finance-p38/` 中的验收文档协同，确保财务模型与实际运营数据一致
