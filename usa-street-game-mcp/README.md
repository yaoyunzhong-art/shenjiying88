# 🎮 USA Street Game MCP — MCP 街机游戏目录

## 📌 目录说明

`usa-street-game-mcp/` 目录存放与 **MCP（Model Context Protocol）街机游戏**相关的配置、模型定义和运营文档。该目录主要用于维护基于 MCP 协议的街机游戏后端逻辑、营收模型配置以及相关参数模板。

## 📂 已有内容索引

| 路径 | 说明 |
|------|------|
| `configs/` | 配置子目录，存放营收模型及参数定义 |
| `configs/revenue-model.md` | 💵 营收模型与预算分配指南，面向单店长的一人运营战斗模型，月预算 $2,000–$5,000 |

### 目录结构

```
usa-street-game-mcp/
├── README.md          ← 本文件（目录说明）
└── configs/
    └── revenue-model.md   # 营收模型配置文件
```

## 🚀 使用指引

1. **查看营收模型**：直接打开 `configs/revenue-model.md` 获取营收分配规则和预算管理策略
2. **扩展配置**：如需添加新的游戏配置或模型参数，请在 `configs/` 下创建对应的 `.md` 或 `.json` 文件
3. **与 MCP 服务协同**：本目录的内容通常通过 MCP 协议被 AI 代理调用和参考，确保各配置文件的格式与协议兼容

## 📝 维护说明

- 所有配置变更应附带版本说明和日期标记
- 保持 `configs/` 目录结构清晰，按功能模块组织子文件
- 本目录属于运营基础设施层，修改前建议确认不影响其他依赖模块
